# Qurantab Landing Page

A responsive landing page built with **HTML and CSS**, inspired by the Qurantab application design.

This project was created as a **pixel-perfect practice**, focusing on translating a real UI design into clean and structured frontend code.

---

## ✨ Overview

The main goal of this project is to practice:

- Building clean and well-structured layouts
- Implementing a real design with high visual accuracy
- Applying responsive design principles
- Improving attention to spacing, alignment, and UI details

The project is intended as a learning exercise and will later be customized into a landing page for a real company.

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox & Grid
- Responsive Design

---

## 🎯 Learning Objectives

- Practice pixel-perfect implementation
- Improve layout and spacing accuracy
- Strengthen understanding of responsive behavior
- Translate UI designs into maintainable code

---

## 📱 Responsive Design

The layout is fully responsive and adapts smoothly to different screen sizes, ensuring a consistent user experience across devices.
